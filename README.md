## 项目介绍

为 workout 应用提供数据和接口支持

前端链接：https://github.com/chenfei2009/workout

所有内容由个人完成，仅供学习交流。

#### 主要功能：

- 训练记录、动作库、计划库、统计数据等对应的接口。

#### 技术栈：

- 客户端：vue
- 服务端：nodejs + express
- 数据库：sql



## 安装

#### 安装依赖

```
npm install
```

#### MySQL配置

修改 [database] 下的数据库参数为正确的配置信息

#### 启动项目

```
node app.js
```
